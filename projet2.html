Projet 2

Le Projet 2 consite à écrire des caractères sur un écran LCD avec un arduino.

Pour ce projet il vous faudra:



-un écran lcd
-un potentiomètres

Le montage est très simple sauf qu'il y a beaucoup de câbles.
Vous pouvez remplacer le potentiomètres par une résistance 1 ohm.

Pour le code il vous suffit de copier de le code et le coller dans l'IDE.Le code est ci dessous:

#include

LiquidCrystal lcd(7,6,5,4,3,2);

void setup() {
lcd.begin(16,2);
}

void loop() {
lcd.setCursor(0,0);
lcd.print("bonjour!");
delay(900);
lcd.clear();


lcd.setCursor(0,1);
lcd.print("salut");
delay(1200);
lcd.clear();

lcd.setCursor(0,0);
lcd.print("tu vas");

lcd.setCursor(0,1);
lcd.print("bien");
delay(1400);
lcd.clear();
}
